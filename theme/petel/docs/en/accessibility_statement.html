Accessibility Statement

The Weizmann Institute of Science places great importance in high quality services and providing equal services to all clients and visitors. For this reason, extensive efforts and resources are put into making the website as accessible as possible.

The purpose of these efforts is to improve the website so that it is accessible, convenient and easy to use for people with motor impairment, vision impairment, people who have cognitive difficulties and for people who have difficulties operating the computer and the internet, thus making it possible for everyone to browse the website easily and quickly while enjoying the content and services developed especially for our clients and visitors.

Standards

The website is compatible with the Internet Explorer 10+, Firefox and Chrome browsers.

The website is compatible with various screen resolutions.

Website content – the website contents have been written in a simple and clear manner.

Website structure – the site structure focuses on clear and convenient navigation with menus comprised of lists, which enable simple, quick and easy orientation throughout the website.

Accessibility Level in the Weizmann Institute of Science Websites

The website was built based on the accessibility guidelines appearing on W3C's Web Content Accessibility Guidelines 2.0, Level AA.

Accessibility Restrictions

Some of the pages are sourced from other websites and therefore did not undergo the accessibility process.

Please note that although we have made great efforts to make all of the website pages accessible, you may discover some parts that are not yet accessible. We continue with our efforts to improve website accessibility as part of our commitment to make the website accessible for everyone including people with disabilities.

Operating Instructions

The “Accessibility” button is located on the upper left side of the page. When you click on this button, a menu will open presenting the various accessibility options available on this website. After selecting the desired accessibility option and clicking on it, the website will reload in the accessible mode. To stop the accessible mode, click on the “Reset” button that appears on the menu after one of the accessibility options has been selected. To open the menu using the keyboard, press the Enter key on the number pad.

Increasing Display Size on the Website

Visitors browsing the website who have a vision impairment and wish to increase the display size on the website can do so by pressing the Ctrl and + keys on the keyboard at the same time (Ctrl+). Every time these keys are pressed at the same time the display size will increase by 10%. In order to reduce the display size, press the Ctrl and – keys on the keyboard at the same time (Ctrl-). In addition, the website enables increasing and decreasing the font size by using the increase/decrease font size buttons located in the accessibility menu.

Operating the Website Using the Keyboard

Visitors browsing the website who are having difficulties operating the mouse can browse the website using the keyboard. Pressing the Tab key repeatedly will move around between the different links on the page. Pressing the Enter key will open the marked link.
